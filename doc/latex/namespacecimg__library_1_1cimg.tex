\subsection{Detailed Description}
Namespace that encompasses {\itshape low-\/level} functions and variables of the C\-Img Library. Most of the functions and variables within this namespace are used by the library for low-\/level processing. Nevertheless, documented variables and functions of this namespace may be used safely in your own source code.

\begin{DoxyWarning}{Warning}
Never write {\ttfamily using namespace \hyperlink{namespacecimg__library_1_1cimg}{cimg\-\_\-library\-::cimg};} in your source code, since a lot of functions of the {\ttfamily cimg\-:\-:} namespace have prototypes similar to standard C functions defined in the global namespace {\ttfamily \-:\-:}. 
\end{DoxyWarning}
